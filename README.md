# fontend

## Build Setup

**Requires Node.js 7+**

``` bash
# install dependencies
npm install # or yarn

# serve in dev mode, with hot reload at localhost:8080
npm run dev

# build for production
npm run build

# serve in production mode
npm start
```

## Mongodb

```bash
//mongodb 后台启动
./mongod -dbpath=/data/db --logpath=/data/mongodb.log --logappend  --port=27017 --fork

```
